# Weather_Diary
**Автор:** *Ш.Илья*
**Язык:** Python 

## Описание программы

Графическое приложение для ведения дневника погоды. Позволяет добавлять записи о погоде за определённый день с указанием температуры, описания и наличия осадков. Данные сохраняются в файл JSON. Доступна фильтрация записей по дате и температуре.

## Требования

- Python 3.7+
- Библиотека Tkinter (входит в стандартную поставку Python)
- Библиотека json (входит в стандартную поставку Python)
- Git

## Как запустить приложение

1. Установите Python с официального сайта (если не установлен).
2. Скачайте или склонируйте этот репозиторий.
3. Откройте терминал в папке проекта.
4. Запустите приложение командой:
   ```bash
   python weather_diary.py
   ```
   или (в зависимости от настройки окружения):
   ```bash
   python3 weather_diary.py
   ```
5. Откроется окно программы.

## Примеры использования

### Добавление записи:
1. Перейдите на вкладку «Добавить запись».
2. Заполните поля:
3. Нажмите «Добавить запись».
4. Запись появится в общем списке.

### Фильтрация по дате:
1. Перейдите на вкладку «Фильтр».
2. Введите дату в поле «Фильтр по дате».
3. Нажмите «Найти».
4. В списке отобразятся только записи за выбранный день.

### Фильтрация по температуре:
1. Перейдите на вкладку «Фильтр».
2. Введите пороговое значение в поле «Фильтр по температуре».
3. Нажмите «Найти».
4. В списке останутся только записи с температурой выше указанного значения.
